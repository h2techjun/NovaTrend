## 📁 프로젝트 개요

NovaTrend는 AI 기반 뉴스 트렌드 대시보드 서비스입니다.

### 핵심 기능

- 📈 글로벌 주식 시장 뉴스 대시보드
- 🪙 암호화폐 뉴스 대시보드
- 🎤 K-POP 아이돌 뉴스 대시보드
- 💬 커뮤니티 토론 게시판

### 기술 스택 (예정)

- **프론트엔드**: Next.js 15 (App Router)
- **백엔드**: Supabase (Auth, DB, Storage)
- **스타일링**: Tailwind CSS + Shadcn UI
- **배포**: Vercel
- **AI**: Google Gemini 2.0 Flash (뉴스 분석/요약)
